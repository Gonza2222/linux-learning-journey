# Manejo de Ficheros en Linux

## Crear archivos y carpetas
- touch → crear archivos vacíos
- mkdir → crear directorios

## Visualizar contenido
- cat → mostrar el contenido de un archivo en la terminal

## Editar archivos
- pico → editor de texto simple en terminal

## Copiar
- cp → copiar archivos o directorios

## Mover o renombrar
- mv → mover o cambiar nombre de archivos/directorios

## Eliminar
- rm → eliminar archivos
- rm -r → eliminar directorios y su contenido

## Reflexión
Aprendí a crear, editar, visualizar y administrar archivos completamente desde la terminal. 
Entendí que el comando rm -r debe utilizarse con cuidado ya que elimina carpetas de forma recursiva.
