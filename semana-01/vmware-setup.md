# Instalación y Configuración de VMware

## Objetivo
Crear un entorno virtual para practicar Linux sin afectar el sistema principal.

## Configuración del equipo
- Windows 10
- 16GB RAM
- 4GB asignados a la máquina virtual
- 2 procesadores
- 40GB de espacio en disco asignados a la maquina virtual
- Ubuntu Desktop como sistema invitado

## Resultado
Máquina virtual funcionando correctamente para prácticas.
